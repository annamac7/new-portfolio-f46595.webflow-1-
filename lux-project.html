Who Is lux?

In Latin, “Lux” translates to “light” → Our brand is about illuminating yourself through content.

  • Empowering, but not entitled
  • Savvy, but not shrewd
  • Passionate, but not intense
  • Convenient, but not awkward

Design Strategy

Site Components:
‍From the start we wanted to make sure that every element that was included in the landing page experience was intentional. We made a list of things the user would be looking for and cross-checked it against a list of features we wanted to include on the homepage to make sure every element was either aiding the user or increasing conversions.

Visual Aesthetic:

  • Make a site that is clean, clear, easy to use and  can be transferred easily to mobile.
  • Dark UI to lessen eye strain for users when streaming.
  • Use of bold, geometric shapes and fonts to keep reader’s attention.
  • Want design to look modern, sleek, and avant-garde while still feeling familiar

Branding

For the branding we had to develop a main brand as well as branding for each of the sub-apps of our platform. To do this, I created a five peg logo that encompasses each of the five sub-apps. Each app was given their own color that can be used for sub-brand identity. The logo resembles a star to play on the "entertainment" aspect.
